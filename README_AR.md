# ONLINE - نسخة بدون ترخيص

هذه نسخة محدثة من تطبيق ONLINE تعمل مباشرة بدون شاشة تفعيل وبدون سيرفر تراخيص.

## ما الذي تغيّر؟

1. تم تعطيل نظام التفعيل الأونلاين بالكامل.
2. التطبيق يفتح مباشرة إلى الصفحة الرئيسية.
3. استقبال رسائل السداد والإرسال التلقائي يعملان بدون ترخيص.
4. بقي اسم التطبيق الظاهر في الهاتف: **ONLINE**.
5. بقيت المزايا السابقة:
   - إدارة الفئات والكروت.
   - إرسال الكرت تلقائيًا عند توفره.
   - إرسال رسالة نفاد الفئة عند عدم توفر الكرت.
   - إدارة المحافظ والأسماء الموثوقة.
   - تعديل رسائل الزبائن.
   - تقارير PDF إجمالية أو لفئة محددة.
   - أيقونة التطبيق المضافة سابقًا.

## رفع النسخة إلى GitHub

1. فك ضغط ملف المشروع.
2. ارفع محتويات المجلد إلى مستودع GitHub، وليس ملف ZIP نفسه.
3. افتح تبويب Actions.
4. شغل Build Android APK.
5. حمّل APK من Artifacts باسم: `online-no-license-apk`.

## ملاحظة

هذه النسخة مخصصة للتجربة والتشغيل بدون نظام اشتراك. عند الرغبة لاحقًا يمكن الرجوع إلى نظام التفعيل الأونلاين أو Google Sheets.


## تحديث النسخ الاحتياطي V10

هذه نسخة ONLINE بدون ترخيص مع إضافة قسم **النسخ الاحتياطي والاستعادة** داخل الإعدادات.

يشمل ملف النسخة الاحتياطية:
- الفئات والكميات.
- كل الكروت المتاحة والمباعة.
- سجلات العمليات.
- المحافظ والأسماء الموثوقة.
- رسائل الزبائن ورسالة نفاد الفئة.
- اسم الشبكة ورقم إدارة الشبكة.
- إعداد تشغيل/إيقاف الإرسال التلقائي.

الاستخدام:
1. قبل حذف التطبيق أو تحديثه: الإعدادات > النسخ الاحتياطي والاستعادة > حفظ نسخة احتياطية.
2. احفظ ملف JSON في الهاتف أو أرسله لنفسك واتساب/تلجرام/جوجل درايف.
3. بعد تثبيت النسخة الجديدة: الإعدادات > النسخ الاحتياطي والاستعادة > استعادة نسخة احتياطية.

ملاحظة: إذا حذفت التطبيق من الهاتف بدون حفظ نسخة احتياطية، قد تضيع البيانات المحلية.


## V11 - تأكيد الحذف

- إضافة نافذة تأكيد قبل حذف الكرت.
- إضافة نافذة تأكيد قبل حذف أي محفظة أو اسم موثوق.
- تحسين نصوص أزرار الحذف لتكون: موافق، حذف / إلغاء.
- استمرار تأكيد حذف الفئات، إشعارات السداد، وحذف كل البيانات.


## تحديث V12 - النسخ الاحتياطي + الاستعادة + تأكيد الحذف + تحسين اختيار الفئة

هذه النسخة تجمع الميزات التالية في تحديث واحد:

1. **النسخ الاحتياطي الكامل** من الإعدادات:
   - الكروت.
   - الفئات.
   - السجلات.
   - المحافظ والأسماء الموثوقة.
   - رسائل الزبائن.
   - اسم الشبكة.
   - رقم إدارة الشبكة.
   - إعدادات الإرسال التلقائي.

2. **استعادة النسخة الاحتياطية** من ملف JSON:
   - تدخل إلى الإعدادات > النسخ الاحتياطي والاستعادة.
   - تضغط: استعادة نسخة احتياطية من ملف.
   - تختار ملف ONLINE_backup_*.json.
   - تظهر رسالة تأكيد قبل الاستبدال.
   - بعد الموافقة يرجع التطبيق البيانات المحفوظة.

3. **تأكيد قبل أي حذف**:
   - حذف كرت.
   - حذف فئة.
   - حذف محفظة أو اسم موثوق.
   - حذف إشعار سداد.
   - حذف كل الإشعارات.
   - حذف كل بيانات التطبيق.

4. **تحسين اختيار الفئة في صفحة استيراد الكروت**:
   - تم استبدال القائمة الباهتة بمربعات فئات واضحة.
   - الفئة المختارة تظهر بإطار بنفسجي واضح.
   - كل فئة تظهر داخل مربع أنيق مثل: 50 ريال، 100 ريال، 200 ريال.

اسم التطبيق: ONLINE.
هذه النسخة بدون ترخيص وتعمل مباشرة بعد منح الصلاحيات.
