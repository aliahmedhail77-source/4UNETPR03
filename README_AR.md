# 4U NET Pro - Clean V1

مشروع جديد من الصفر لإرسال الكروت تلقائيًا عند وصول رسائل الإيداع.

## المبدأ

هذه نسخة تأسيسية نظيفة. أي تعديل مستقبلي يكون تحديثًا عليها، وليس على النسخ القديمة.

## الميزات الموجودة في V1

- واجهة For You حديثة.
- فئات افتراضية:
  50، 100، 150، 200، 250، 300، 500.
- إضافة فئات جديدة.
- إدارة الفئات: إضافة، تعديل، حذف، تفعيل/إيقاف.
- عرض الكروت داخل كل فئة.
- تعديل كرت.
- حذف كرت.
- استيراد كروت من TXT.
- إضافة كروت يدويًا.
- قراءة رسائل:
  - Jawali / جوالي
  - Jaib / جيب
  - ONE Cash / ون كاش
- في ONE Cash يتم الاعتماد على الأسماء الموثوقة ومطابقة الاسم الثلاثي.
- إرسال كرت من نفس الفئة فقط.
- سجلات العمليات.
- صفحة تقارير تمهيدية، وتصدير PDF سيضاف في V2.

## بناء APK عبر GitHub Actions

بعد رفع المشروع إلى GitHub:

1. افتح تبويب Actions.
2. شغل Build Android APK.
3. بعد النجاح حمّل Artifact:
   4unet-pro-clean-v1-apk
4. فك الضغط وثبّت app-debug.apk.


## تحديث V2 - التقارير والإشعارات

تمت إضافة المتطلبات التالية:

1. تنزيل تقرير PDF إجمالي لكل الفئات من صفحة السجلات.
2. تنزيل تقرير PDF لفئة محددة مثل 100 أو 200 أو 500 ريال.
3. حذف إشعار سداد واحد من السجل، أو حذف كل إشعارات السداد.
   - ملاحظة مهمة: حذف الإشعار لا يعيد الكرت إلى المتاح حتى لا يحدث بيع مكرر لنفس الكرت.
4. إشعار داخل نظام Android عند بيع كل كرت بنجاح.
5. إشعار تنبيه عند فشل إرسال SMS بعد حجز الكرت.
6. تعديل نص الرسالة المرسلة للمشترك إلى الصيغة المطلوبة:

تم استلام 100ريال
رقم الكرت: 1547013174
لشبكة: فور يو نت
فئة 100ريال

7. تعديل اسم التطبيق والواجهة الرئيسية إلى: حاسب اونلاين.


## تحديث V3 - التقارير ونصوص الرسائل

- إضافة تقرير PDF إجمالي مفصل يقسم البيانات إلى مربعات مستقلة لكل فئة.
- إضافة تقرير PDF مختصر إجمالي.
- عند اختيار فئة محددة، يمكن اختيار تقرير مفصل أو تقرير مختصر للفئة فقط.
- كل مربع فئة يحتوي: إجمالي الكروت، المباعة، المتبقية، عمليات الإرسال الناجحة، وإجمالي مبلغ الفئة.
- إضافة صلاحية تعديل نص رسالة إرسال الكرت من الإعدادات.
- إضافة صلاحية تعديل نص رسالة نفاد الفئة من الإعدادات.
- المتغيرات المتاحة في الرسائل: {amount} للمبلغ، {card} لرقم الكرت، {network} لاسم الشبكة.

نموذج رسالة الكرت الافتراضية:
تم استلام {amount}ريال
رقم الكرت: {card}
لشبكة: {network}
فئة {amount}ريال

نموذج رسالة نفاد الفئة الافتراضية:
تنبيه: تم استلام {amount}ريال لكن لا توجد كروت متاحة لفئة {amount}ريال.
رقم إدارة الشبكة: 776901570

## تحديث V4 - مركز تعديل الرسائل

هذا التحديث مبني مباشرة على V3، وليس مشروعًا جديدًا.

تم تحسين إدارة الرسائل لتصبح أوضح وأسهل:

- إضافة صفحة مستقلة باسم: مركز تعديل رسائل الزبائن.
- عرض معاينة مباشرة لرسالة إرسال الكرت قبل الحفظ.
- عرض معاينة مباشرة لرسالة نفاد الفئة قبل الحفظ.
- إمكانية تعديل رسالة إرسال الكرت بشكل كامل وإضافة عروض أو ملاحظات.
- إمكانية تعديل رسالة نفاد الفئة بشكل كامل.
- إضافة زر سريع لإضافة سطر عرض داخل رسالة إرسال الكرت.
- إضافة زر تحديث المعاينة داخل نافذة التعديل.
- إضافة زر استعادة النص الافتراضي لكل رسالة على حدة.
- إضافة تعديل اسم الشبكة من داخل مركز الرسائل.
- إضافة تعديل رقم إدارة الشبكة من داخل مركز الرسائل.
- إضافة المتغير الجديد: {adminPhone} لرقم إدارة الشبكة.

المتغيرات المدعومة الآن:

{amount} = مبلغ الفئة
{card} = رقم الكرت
{network} = اسم الشبكة
{adminPhone} = رقم إدارة الشبكة

نموذج رسالة الكرت الافتراضية:

تم استلام {amount}ريال
رقم الكرت: {card}
لشبكة: {network}
فئة {amount}ريال

يمكنك إضافة عرض مثل:

عرض خاص: اشترِ كرتين واحصل على خصم في المرة القادمة.

نموذج رسالة نفاد الفئة الافتراضية:

تنبيه: تم استلام {amount}ريال لكن لا توجد كروت متاحة لفئة {amount}ريال.
رقم إدارة الشبكة: {adminPhone}
